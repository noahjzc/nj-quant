## 策略逻辑

### MA均线策略

- **买入条件**：MA5从下方穿过MA20（金叉）
- **卖出条件**：MA5从上方穿过MA20（死叉）
- **持仓周期**：中期（数周到数月）

### MACD策略

- **买入条件**：DIF从下方穿过DEA（金叉）
- **卖出条件**：DIF从上方穿过DEA（死叉）
- **持仓周期**：中期趋势跟踪

## 回测流程

1. 加载CSV数据（指定股票）
2. 数据预处理（按日期排序、去除空值）
3. 计算技术指标（数据已预计算，直接使用）
4. 生成交易信号（金叉/死叉判定）
5. 模拟交易（买入/卖出/持仓计算）
6. 输出结果

## 回测结果输出

每只股票输出：

- **交易明细表**：日期、信号类型(买入/卖出)、价格、收益率
- **最终统计**：
    - 总收益率
    - 年化收益率
    - 最大回撤
    - 交易次数
    - 胜率
    - 同期指数对比

## 技术实现

### 交易信号判定

```python
# 金叉：MA5上穿MA20
buy_signal = (ma5 > ma20) & (ma5.shift(1) <= ma20.shift(1))

# 死叉：MA5下穿MA20
sell_signal = (ma5 < ma20) & (ma5.shift(1) >= ma20.shift(1))
```

### 收益率计算

- 单次交易收益率 = (卖出价 - 买入价) / 买入价
- 总收益率 = 所有盈利交易之和 - 所有亏损交易之和

## 新增功能 (2026-04-19)

### 1. 风险管理 - 止损/止盈

BacktestEngine now supports:
- `stop_loss`: 止损阈值，如0.10表示亏损10%时自动卖出
- `take_profit`: 止盈阈值，如0.20表示盈利20%时自动卖出

交易记录中新增 `stop_loss_triggered` 和 `take_profit_triggered` 字段。

### 2. 组合策略

CombinedStrategy 结合趋势跟踪和均值回归：
- 买入：MA5上穿MA20（金叉）且 RSI < 40
- 卖出：MA5下穿MA20（死叉）或 RSI > 60

### 3. 参数优化器

PortfolioOptimizer 对策略参数进行网格搜索，找到历史表现最佳的参数组合。

### 4. 组合回测

PortfolioBacktest 支持多股票等权分配资金回测，汇总整体表现。

### 5. 可视化

Visualizer 生成资金曲线图、买卖点标记图、策略对比图。

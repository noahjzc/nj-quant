## 策略逻辑

### MA均线策略
- **买入条件**：MA5从下方穿过MA20（金叉）
- **卖出条件**：MA5从上方穿过MA20（死叉）
- **持仓周期**：中期（数周到数月）

### MACD策略
- **买入条件**：DIF从下方穿过DEA（金叉）
- **卖出条件**：DIF从上方穿过DEA（死叉）
- **持仓周期**：中期趋势跟踪

## 回测流程

1. 加载CSV数据（指定股票）
2. 数据预处理（按日期排序、去除空值）
3. 计算技术指标（数据已预计算，直接使用）
4. 生成交易信号（金叉/死叉判定）
5. 模拟交易（买入/卖出/持仓计算）
6. 输出结果

## 回测结果输出

每只股票输出：
- **交易明细表**：日期、信号类型(买入/卖出)、价格、收益率
- **最终统计**：
  - 总收益率
  - 年化收益率
  - 最大回撤
  - 交易次数
  - 胜率

## 技术实现

### 交易信号判定
```python
# 金叉：MA5上穿MA20
buy_signal = (ma5 > ma20) & (ma5.shift(1) <= ma20.shift(1))

# 死叉：MA5下穿MA20
sell_signal = (ma5 < ma20) & (ma5.shift(1) >= ma20.shift(1))
```

### 收益率计算
- 单次交易收益率 = (卖出价 - 买入价) / 买入价
- 总收益率 = 所有盈利交易之和 - 所有亏损交易之和

# 多策略综合评分量化选股系统设计

## 背景问题

当前系统问题：
- 信号强度计算与策略评估方式不匹配
- MACD策略在信号评分中占优势（上涨趋势中DIF-DEA差值持续为正）
- RSI/KDJ等均值回归策略被低估（强势股评分反而低）
- 导致策略轮动变成"单策略轮动"，失去了多策略分散风险的意义

## 目标

实现**多策略综合加权评分**系统：
- 不依赖单一策略
- 综合考虑MACD、MA、RSI、KDJ、成交量等多个维度
- 根据市场状态动态调整权重

## 核心设计

### 综合评分公式

```
综合得分 = MACD得分×35% + MA得分×20% + RSI得分×15% + KDJ得分×15% + 成交量得分×15%
```

### 各策略信号强度计算改进

#### RSI信号强度
- **超卖程度**：RSI < 30 时越低越强
- **反弹动量**：RSI转头向上的幅度
- **综合**：超卖程度×0.6 + 反弹动量×0.4

#### KDJ信号强度
- **超卖程度**：J < 20 时越低越强
- **反弹动量**：J值从低点回升的幅度
- **综合**：超卖程度×0.6 + 反弹动量×0.4

#### 布林带信号强度
- **偏离程度**：价格偏离中轨的百分比
- **突破方向**：价格由下往上穿越中轨时更强

#### MA信号强度
- **开口角度**：(MA5-MA20)/MA20 的变化率
- **多头排列**：MA5 > MA20 > MA60 时更强

### 权重配置

| 策略 | 权重 | 原因 |
|------|------|------|
| MACD | 35% | 趋势确认能力强，最能捕捉主升浪 |
| MA | 20% | 确认中期趋势方向 |
| RSI | 15% | 超跌反弹参考 |
| KDJ | 15% | 灵敏度高，辅助判断 |
| 成交量 | 15% | 资金推动确认 |

### 动态权重调整（可选增强）

根据市场状态调整权重：
- **强势市场**（沪深300近期涨幅>5%）：提高MACD/MA权重
- **弱势市场**（沪深300近期跌幅>5%）：提高RSI/KDJ权重

## 参数设置

| 参数 | 值 | 说明 |
|------|-----|------|
| 持仓数量 | 5只 | 每周持仓股票数 |
| 持仓权重 | 20% | 等权重分配 |
| 调仓频率 | 周度 | 每周五收盘后换仓 |
| 初始资金 | 100万 | 基准回测资金 |

## 输出

- 每周综合评分最高的5只股票
- 各股票的综合评分明细
- 与单策略轮动的对比

## 回测对比

同时运行两种回测进行对比：
1. **原策略轮动**：单策略选择 + 单策略信号选股
2. **综合评分版**：多策略综合评分选股

评估指标：
- 总收益率
- 年化收益率
- 最大回撤
- 夏普比率
- 策略稳定性（每周变化程度）

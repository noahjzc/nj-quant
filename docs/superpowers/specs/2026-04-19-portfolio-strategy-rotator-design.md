# 策略轮动量化选股系统设计

## 目标

构建一个**策略轮动+量化选股**的组合投资系统：

- 每周从全市场5711只A股中动态筛选5只股票
- 每周根据过去4周表现自动选择最优策略
- 等权重持仓（每只20%），周五尾盘换仓

## 核心流程

```
┌─────────────────────────────────────────────────────────────┐
│  每周循环（周五收盘后执行）：                                    │
│                                                             │
│  1. 策略评估                                                 │
│     - 对13种策略在过去4周的模拟表现评分                          │
│     - 计算每种策略的4周累计收益率                               │
│                                                             │
│  2. 策略选择                                                 │
│     - 选出过去4周累计收益率最高的策略                           │
│                                                             │
│  3. 股票筛选                                                 │
│     - 对5711只股票应用选中策略                                 │
│     - 计算每只股票的技术指标信号强度                            │
│                                                             │
│  4. 选股排序                                                 │
│     - 按信号强度排序，选取前5只                                 │
│                                                             │
│  5. 等权持仓                                                 │
│     - 每只股票分配20%仓位                                      │
│     - 下周一开盘买入，周五尾盘卖出                              │
└─────────────────────────────────────────────────────────────┘
```

## 参数设置

| 参数     | 值    | 说明          |
|--------|------|-------------|
| 持仓数量   | 5只   | 每周持仓股票数     |
| 持仓权重   | 20%  | 等权重分配       |
| 调仓频率   | 周度   | 每周五收盘后换仓    |
| 策略评估周期 | 4周   | 用过去4周表现评估策略 |
| 初始资金   | 100万 | 基准回测资金      |

## 策略列表

系统支持13种策略：

| 策略名称    | 类型    | 核心逻辑              |
|---------|-------|-------------------|
| MA均线策略  | 趋势跟踪  | MA5/MA20金叉买入，死叉卖出 |
| MACD策略  | 趋势跟踪  | DIF/DEA金叉买入，死叉卖出  |
| RSI策略   | 均值回归  | RSI<30买入，RSI>50卖出 |
| 组合策略    | 混合    | MA+RSI共振          |
| 布林带策略   | 通道突破  | 价格触及下轨买入，中轨卖出     |
| 布林带严格版  | 通道突破  | 严格版，减少假信号         |
| KDJ超卖策略 | 超卖反弹  | J值超卖后反弹买入         |
| KDJ金叉策略 | 趋势跟踪  | KDJ低位金叉买入         |
| 多周期RSI  | RSI共振 | 3周期RSI共振买入        |
| RSI多周期版 | RSI共振 | 更严格的多周期共振         |
| 趋势确认策略  | 趋势跟踪  | MA多头排列+MACD金叉     |
| 趋势回踩策略  | 趋势跟踪  | 回踩MA20后买入         |
| 成交量异常策略 | 资金推动  | 放量突破买入            |
| 放量确认策略  | 混合    | 趋势+放量确认           |

## 信号强度计算

每种策略的信号强度定义：

| 策略类型  | 信号强度指标           |
|-------|------------------|
| RSI类  | RSI值越低强度越高（超卖程度） |
| MACD类 | DIF与DEA差值越大强度越高  |
| MA类   | 均线开口角度越大强度越高     |
| KDJ类  | J值越低（超卖）强度越高     |
| 布林带类  | 价格偏离中轨越远强度越高     |
| 成交量类  | 量比越大强度越高         |

## 策略评估逻辑

对每种策略在过去4周的表现评分：

```
策略得分 = Σ(每只股票在该策略下的4周收益率) / 股票数量
```

评分后选择得分最高的策略作为下周主策略。

## 回测流程

1. 从2020-01-01开始回测
2. 每周五执行策略评估和选股
3. 下周一开盘买入选中股票
4. 周五收盘前卖出（以收盘价计算）
5. 记录每周净值、持仓变化、收益明细

## 输出指标

- 总收益率
- 年化收益率
- 最大回撤
- 夏普比率
- 每周持仓明细
- 策略切换记录

## 架构设计

```
├── strategy_evaluator.py     # 策略评估器：计算各策略历史表现

├── stock_selector.py          # 选股器：根据信号强度排序选股
├── portfolio_builder.py       # 组合构建器：生成最终持仓
└── backtest_runner.py         # 回测运行器：执行回测并输出结果
```

## 风险提示

- 历史回测不代表未来收益
- 全市场筛选计算量大，需优化性能
- 策略轮动可能频繁切换，增加交易成本
- 未考虑流动性风险（停牌、涨跌停）
- 信号强度计算方式可能需要根据实际效果调整
